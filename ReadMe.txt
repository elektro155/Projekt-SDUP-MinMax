Opis folderów:
GUI sources - kody źródłowe do aplikacji Desktopowej Windows Forms obsługującej komendy na UART
HORNER IP - kody źródłowe w języku Verilog dla modułu IP liczącego algorytm Hornera
Komunikacja zdjęcia - zrzuty ekranu komunikacji płyty ZedBoard z komputerme przez UART na kosoli i przez program GUI
Testbench - kod źródłowy testu w języku Verilog oraz zrzut ekranu przebiegów czasowych
ZYNQ ARM - kod żródłowy w języku C na procesor ARM w układzie ZYNQ
